# Vestibular Frontend

Versão Frontend do sistema de vestibular

## Começando

Clone este projeto e execute o comando abaixo:

```
  npm install && npm start
```

### Pré-requisitos

Backend [https://github.com/lelodois/vestibular-backend]

```
  NodeJs
  Npm
  AngularCli
```
### Tecnologias

* Angular 4
* Bootstrap
* Typescript
