## Vestibular Frontend

Versão Frontend do sistema de vestibular

* ** Backend ** - *Java API* - [Source code](https://github.com/lelodois/vestibular-backend)

### Começando

Clone este projeto e execute o comando abaixo:

```
  npm install && npm start
```

### Pré-requisitos

```
  NodeJs
  Npm
  AngularCli
```
### Tecnologias

* Angular 4
* Bootstrap 3
* Typescript
* JSON Web Tokens
